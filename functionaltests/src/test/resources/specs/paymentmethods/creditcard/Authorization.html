Credit Card Authorization

The integration service allows to reserve an amount of money to be charged later.

Note: it is necessary to do a creditcardcheck first via PAYONE client API to avoid complying to PCI-DSS standard.

Examples

Common Setup

  • payment.paymentMethodInfo.method: CREDITCARD
  • payment.transactions[0].type: AUTHORIZATION
  • payment.amountPlanned.centAmount: 2000
  • payment.amountPlanned.currencyCode: EUR

Successful Authorization

If an authorization for a payment is successfully requested, then

  • only 1 interfaceInteraction was added
  • for a PAYONE preauthorization request
  • and a final state ofSUCCESSFUL
