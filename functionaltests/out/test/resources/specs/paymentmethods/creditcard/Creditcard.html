Credit Card Payments

The integration service supports credit card payments.

Note: It is necessary to request a creditcardcheck via PAYONE's client API to avoid complying to PCI-DSS standard.

Checkout Processes

  • Authorization only (without 3-D Secure)
  • Authorization only (verified by 3-D Secure)
  • Charge immediately (without separate Authorization, without 3-D Secure)
  • Charge immediately (without separate Authorization, verified by 3-D Secure)

Post Purchase Processes
